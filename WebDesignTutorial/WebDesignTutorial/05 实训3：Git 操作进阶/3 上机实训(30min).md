## 2.1【实操目的】

1. 学会利用 `git status` 命令查看上一次 commit 之后仓库中文档的变化。
2. 学会利用 `git add .` 命令一次性确认多个文件的更新。
3. 学会利用 `git reset` 命令切换仓库中的文档版本。
4. 学会利用 `git reflog` 命令查看版本仓库的历史版本变化日志，并找到版本回退后消失的那部分更新的 id。
